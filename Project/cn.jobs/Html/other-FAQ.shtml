  1. 如何申请职位？

    您需要注册并完成基本简历的填写，就可看到cn.jobs上所有的职位

  2. 我已经在cn.jbs注册，为什么在申请的时候还需要再注册？

    招聘中国保留您的简历是为了方便企业查找以及向您推送适合您的职位信息，本网站收集的职位大多来源于全球500强企业，它们都有独立的招聘管理系统，所以需要您重新填写。

  3. 个人是否可以申请.jobs域

    .jobs域名只向企业端开放，个人不能申请名

  4. JOBS是国际顶级域名，和别的国际顶级域名有什么区别吗？

    和edu只用于教育，gov只用于政府一样，JOBS是只用于招聘的，所有申请者必须是企业，不能是中介，需要认证，这和不需要认证的com等域名有很大不同。除此之外，JOBS和其他域名最大的区别，就是由Direct Employers 和中广互联等公司提供的增值服务。 每一个拥有JOBS域名的企业，其职位会自动同步到相关的地区性和行业性网站。例如，杭州的一个化工厂，只要使用JOBS域名，该公司的所有职位就会自动同步到www.hangzhou.jobs和www.huagong.jobs等国内网站， 根据需要，也可以自动同步到国外相关网站。

  1. 企业申请.jobs域名需要提供哪些资料？

    请您点击网页右上方的“域名申请”填写您的联系方式，我们的销售人员会在24小时之内与您取得联系，并向你核实企业的相关信息，包括营业执照、企业机构代码等。

  2. 我们公司想申请地区性的网站，如：合肥(www.hefei.jobs)，请问如何申请？

    原则上地区性域名不对外开放，由中广互联独家经营，但我们欢迎第三方合作，您可以发送邮件到info@cn.jobs，也可电话咨询400-877-6880

  4. 我们公司没有官网，或者我们的官网没有招聘页面，那怎么使用JOBS的同步功能？

    中广互联（北京）科技有限公司根据Direct Employers长期服务国际大公司的经验，将为您的企业量身打造属于自己的招聘微网站。该网站将可以发布职位，自动同步，同时还可以定制同步到其他招聘网站，譬如58同城、前程无忧、新浪微博等，让您的职位发布更方便，简历管理更简单。同时网站集成信息展示功能，您可以自己编写“公司介绍|”、“员工生活等”、“工作环境”等内容，让应聘者更完整的了解贵公司。除此之外，该微网站会使用贵公司的LOGO，综合考虑配色等因素，从外观上体现出贵公司的文化气息。请参考我们给希尔顿集团定制的某个微网站http://ihg-disability.jobs/
